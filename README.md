# QAtest_Avito
Найденные баги
High  
1. Фильтр “Команда” говорит «Вакансий пока нет», но вакансии отображаются
Тип: логический баг
Почему high: противоречие ключевой функции фильтрации; вводит в заблуждение пользователя.

2. Фильтр «Можно удалённо» показывает вакансии с пометкой «офис»
Тип: логический баг
Почему high: ключевой фильтр не работает; критическое нарушение функционала поиска.

3. В категории «Клиентский сервис» указано 2 вакансии, но отображается 1
Тип: несоответствие данных
Почему high: неверный подсчёт элементов в категории, создаёт ложные ожидания.

Medium  
4. Выбрана специализация «Data Science», но вакансии размазаны по другим категориям
Тип: логический
Почему medium: снижает точность поиска, но не полностью ломает выдачу.

5. Активный тег «Поддержка пользователей», но выдача ему не соответствует
Тип: логический
Почему medium: фильтрация работает неправильно, но не критично для навигации.

6. Несовместимые фильтры применены одновременно, но выдача некорректная
Тип: логический
Почему medium: некорректное объединение фильтров → неправильная логика.

7. «227 вакансий открыто» не соответствует отображаемому перечню
Тип: несоответствие данных
Почему medium: может вводить в заблуждение, но не влияет на функционал фильтров напрямую.

8. Сообщение «Ничего не нашлось» показано в середине списка, затем следуют вакансии
Тип: UX / логический
Почему medium: сбивает с толку, нарушает порядок отображения результатов.

Low  
9. Неконсистентные отступы между блоками
Тип: UI
Почему low: влияет только на внешний вид.

10. Внутренние теги вакансий визуально похожи на фильтры
Тип: UX
Почему low: путаница интерфейса, но функционал не нарушен.

11. Иконки “X” у тегов расположены неравномерно
Тип: UI
Почему low: проблема оформления.

12. Поле ввода имеет слишком большой внутренний отступ слева
Тип: UI
Почему low: чисто визуальный дефект.
